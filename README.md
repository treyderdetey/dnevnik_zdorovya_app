# Ассистент Здоровья

Современное мобильное приложение на платформе Flutter для комплексного мониторинга показателей здоровья, управления привычками и медицинского контроля.

## 🌟 Основные функциональные модули

*   **💧 Мониторинг гидратации**: Интеллектуальный трекер потребления воды с системой персональных напоминаний.
*   **💊 Фармакологический контроль**: Учет приема лекарственных препаратов с гибким расписанием и подтверждением выполнения.
*   **🩺 Женское здоровье**: Прогнозирование циклов, логирование симптомов и мониторинг физиологических показателей.
*   **😴 Трекер качества сна**: Анализ продолжительности отдыха и формирование здоровых привычек сна.
*   **🎭 Психоэмоциональный дневник**: Отслеживание настроения и анализ влияния внешних факторов на самочувствие.
*   **📈 Аналитическая панель**: Интерактивные графики показателей и экспорт детальных медицинских отчетов в формате PDF.
*   **🏆 Система геймификации**: Адаптивная система достижений (Rewards), мотивирующая пользователя на соблюдение режима.

## 🛠 Технологический стек

*   **Frontend**: Flutter SDK (Dart) — кроссплатформенная разработка.
*   **Architecture**: Clean Architecture + Provider (State Management).
*   **Database**: Hive (NoSQL) — высокопроизводительное локальное хранилище с бинарной сериализацией.
*   **System Tools**: Gradle (Android Build System) — конфигурация сборки и управление зависимостями.
*   **Notifications**: Flutter Local Notifications — работа с системными каналами уведомлений Android/iOS.
*   **Analytics & UI**: FL Chart (визуализация), Google Fonts, Flutter Animate.

## 🔒 Безопасность и Приватность

*   **Локальность**: Приложение работает по принципу "Local-First". Персональные данные не передаются на сторонние сервера и хранятся исключительно в защищенной песочнице (Sandbox) устройства.
*   **Аппаратная изоляция**: Использование системных механизмов Android для разграничения прав доступа.

## 📦 Развертывание и сборка

1.  **Требования**: Flutter SDK >= 3.2.0.
2.  **Загрузка зависимостей**:
    ```bash
    flutter pub get
    ```
3.  **Генерация адаптеров БД**:
    ```bash
    flutter pub run build_runner build --delete-conflicting-outputs
    ```
4.  **Сборка проекта**:
    ```bash
    # Для разработки
    flutter run
    # Для создания релизной версии
    flutter build apk --release
    ```
